get_counting_events¶

get_counting_events(contador_nombre, contador_línea_tpl, seguimientos, incidencia_mínima, cls_to_idx={0: 0, 1: 2, 2: 4})¶

Generates counting events based on the intersection of segments with a counting line and an incidence angle.

This function iterates through tracked objects and detects when they cross a defined line within a specified range of angles, indicating a counting event. Each event is yielded with detailed information for further processing.

Parameters:

  • contador_nombre (str) – The name of the counter used to identify the set of events.

  • contador_línea_tpl (tuple) – Starting and ending coordinates of the counting line (x1, y1, x2, y2).

  • seguimientos (pandas.DataFrame) – A pandas DataFrame containing the tracks of objects to be counted.

  • incidencia_mínima (float) – The minimum incidence angle required to count a crossing.

  • cls_to_idx (dict, optional) – A dictionary mapping object classes to specific indices. Defaults to {0: 0, 1: 2, 2: 4}.

Yields:

A tuple containing another tuple with frame information and detections, and a second one with tracking details.
